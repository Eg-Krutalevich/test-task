# test-task

Репозиторий содержит решения задач по следующим темам: условные, операторы, массивы. В директории ```src/test``` также находятся тесты по некоторым задачам по теме «Условные операторы»
# Используемые технологии

* Java 17
* JUnit 5
* Allure
